# DTPennyRoute

Mobile-friendly web app for Dollar Tree penny hunting.

Build 9 focuses the app around the correct flow:

1. Set ZIP
2. Add penny items to My Run
3. Build Route
4. Log stock/penny results inside the app
5. Worth the Drive score ranks stores

External Dollar Tree search links were removed because they can open with the wrong ZIP. The live Dollar Tree stock lookup is intentionally separated as a future backend integration.
