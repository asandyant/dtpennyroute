# DTPennyRoute

Mobile-friendly web app for Dollar Tree penny hunting.

Build 4 Complete Base includes:
- Dashboard
- ZIP code setting
- Login/sign up foundation
- Penny List search
- My Run saved list
- Store Check workflow
- Community report feed
- Worth the Drive Score
- Admin add-item screen
- Render-ready Node/Express backend

Deploy:
- Build Command: npm install
- Start Command: npm start
